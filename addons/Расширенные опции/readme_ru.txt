Модуль "Связанные опции" позволяет указывать для товаров доступные комбинации опций (связывать опции).

Это может быть полезно при продаже товаров, имеющих взаимозависимые опции (характеристики), например цвет и размер у одежды.
После установки модуля, при редактировании товаров отображается закладка "Связанные опции", на которой можно задать доступные комбинации опций для товара.

Основные возможности модуля:

 - комбинировать/связывать любое количество опций (одновременно для товаров можно использовать обычные, не связанные опции)
 - использовать для связи опции типов "Список" ("select"), "Переключатель" ("radio") и "Изображение" ("image")
 - задавать различные варианты связанных опций для разных товаров (цвет+размер для одних товаров, длина+ширина для других и т.д.)
 - указывать разные цены (в т.ч. скидки, акции), веса и модели, артикулы для разных комбинаций связанных опций
 - собирать код товара (модель) по частям: базовый код товара + код выбранной комбинации связанных опций
 - ограничить выбор покупателя только доступными комбинациями опций (другие значения либо скрываются, либо делаются недоступными)
 - позволять покупателю выбирать значения связанных опции в любом порядке, все ограничения сразу распространятся на другие опции
 - использовать пошаговый выбор опций - покупатель должен выбрать сначала первую, опцию, затем вторую и т.д. (опционально)
 - вести учет количества в разрезе наборов связанных опций
 - добавить для покупателя на страницу товара кнопку "Очистить параметры", очищающую все значения опций (опционально)
 - указать комбинацию опций выбранную "по-умолчанию" (опционально)
 - импорт/экспорт в формате XLS (требуется PHPExcel)
 - полная совместимость с модулями обновления цен "Живая цена" https://liveopencart.ru/opencart-moduli-shablony/moduli/tsenyi/jivaya-tsena-dinamicheskoe-obnovlenie-tsenyi-2 и "Живая цена PRO" https://liveopencart.ru/opencart-moduli-shablony/moduli/tsenyi/jivaya-tsena-pro-dinamicheskoe-obnovlenie-tsenyi

Совместимость: OpenCart 2.X

Установка:
 - Скопируйте содержимое папки upload на сайт (в корневую директорию opencart). Ваши файлы перезаписаны не будут.
 - Выполните установку модуля через меню "Дополнения / Модули" - "LIVEOPENCART: Связанные опции" - "Установить"
 - Настройте необходимые варианты связанных опций через меню "Дополнения / Модули" - "LIVEOPENCART: Связанные опции" - "Изменить"
 
Обновление версии модуля (не рекомендуется обновлять модуль, если у Вас нет необходимости в добавленном функционале):
 - Скопируйте содержимое папки upload на сайт с перезаписью существующих файлов (в корневую директорию opencart).
 - Выполните установку модуля через меню "Дополнения / Модули" - "LIVEOPENCART: Связанные опции" - "Изменить" (будет отображено сообщение об обновлении версии модуля)
